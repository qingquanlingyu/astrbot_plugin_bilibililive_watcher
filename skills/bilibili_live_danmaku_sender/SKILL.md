# Bilibili Live Danmaku Sender

## 前置条件

先确认这些条件成立：
- 已安装并配置 astrbot_plugin_bilibililive_watcher 连接插件
- 上下文中存在`<bili_live_room_state>`块，且内部状态为"直播中"
- 用户明确表达了“替我发一条弹幕”“帮我回一句到直播间”“把这句话发到直播间”这类代发意图

## 典型可调用场景
- “帮我发一句：主播今天状态真好”
- “发个弹幕，夸一下这首歌好听”
- “把‘这波操作太秀了’发到直播间”

## 不应调用的场景
- 用户只是询问直播间发生了什么
- 用户只是让你总结弹幕、分析气氛、解释主播刚才说的话
- 用户没有明确要求代发，只是在讨论“可以发什么”
- 同一轮已经调用过一次发送工具，不要重复发送

## 调用方法

使用工具`bili_live_send_danmaku`发送最终要发出的那一句弹幕

调用要求：
- 单轮最多调用一次
- 调用前先把要发送的话组织成最终版本，再调用，不要边试边发
- 工具失败后不要在同一轮自动重试，也不要改写一句类似的话再次发送
- 调用完成后，向用户简短说明已发送或发送失败即可
